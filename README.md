# What is it?
This is command line java password generator. The basic function is to generate a password for the user per their requests. The program initially prompts the user for the desired length of their password, it must be between 12 and 20 characters to be valid. The program then asks the user what to include in the the password letter pool, first asking if letters should be included,then numbers and symbols. These charachters are then added to the pool of possible characters and then iterated over to create a strong password.
